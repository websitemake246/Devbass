Authentication

Include header: x-api-key: YOUR_API_KEY

Endpoints

POST /api/data - Store data
Body: {"key": "myKey", "value": {...}}

GET /api/data/:key - Retrieve data

POST /api/upload - Upload file (multipart/form-data)
Fields: file (binary), filename (optional)

GET /api/file/:filename - Download/view file

DELETE /api/data/:key - Delete data

cURL Examples

# Store JSON curl -X POST https://yourdomain.com/api/data \ -H "x-api-key: YOUR_KEY" \ -H "Content-Type: application/json" \ -d '{"key":"profile","value":{"name":"John"}}' # Get data curl -X GET https://yourdomain.com/api/data/profile \ -H "x-api-key: YOUR_KEY" # Upload file curl -X POST https://yourdomain.com/api/upload \ -H "x-api-key: YOUR_KEY" \ -F "file=@image.jpg"
